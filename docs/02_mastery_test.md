## B. 수식 유도 (쓸 수 있어야 함)

**B1.** Eq(2): RV_{i,t} = Σ_{j=0}^{20} r²_{i,d_t−j} — 월말 직전 21거래일 제곱합. 왜 GARCH가 아닌가? → 모델 없이 관측된 위험을 직접 측정, 각주 11: 자기상관 보정해도 결과 불변.
**B2.** Eq(3)+OOS: RV_t = α + ρRV_{t−1} + ε. OOS R² = 1 − Σ(실제−AR예측)²/Σ(실제−과거평균)². 초기 240개월 학습, 확장윈도우. 벤치마크가 "과거평균"인 이유 → 예측이 무정보 대안보다 나은가를 잰다.
**B3.** Eq(5): σ̂²_t = 21·Σ_{j=0}^{125} r²_{d_{t−1}−j}/126 — 6개월(126일) 일평균분산×21=월분산. **d_{t−1}**: t월 스케일에 t−1월말까지만 → look-ahead 없음.
**B4.** Eq(6): r_{WML*,t} = (σ_target/σ̂_t)·r_{WML,t}, σ_target=연 12%. σ_target이 샤프·IR에 불변인 이유 → 전 시계열 상수배는 평균·표준편차를 같은 비율로 바꿈.
**B5.** Eq(7): RV_wml = β²·RV_rmrf + σ²_e (6개월 일별로 β·분산 추정). 이 분해가 논문의 지적 정점인 이유 → β 헤지 실패의 정량적 설명.
**B6.** Eq(8)–(10) 회전율: x_t = 0.5Σ|w_{i,t} − w̃_{i,t−1}|, w̃=가격표류 반영 직전 가중치, WML*는 L_t=σ_target/σ̂_t로 나눠 스케일 반영.

## C. 추론 (왜 되는가)

**C1.** 왜 폭락은 "시장이 큰 하락 뒤 반등할 때" 터지나? → 곰장에서 승자=저β·패자=고β가 되어 WML의 β가 음(−)으로. 반등이 오면 숏(고β 패자)이 폭등 — 1998-10 한국: 패자 +121% vs 승자 +26%.
**C2.** 왜 β 헤지는 실패하고 실현변동성 스케일링은 되나? → 시장성분은 총위험의 23%뿐 + β²는 거의 예측불가(5.33%). 고유성분이 크고(77%) 잘 예측됨(47.06%). 스케일링은 총위험(=주로 고유위험)을 직접 겨냥.
**C3.** variable + predictable 중 하나만 성립하면? → variable만: 스케일이 노이즈 추종, 소용없음. predictable만(상수면): 스케일이 상수 — 할 게 없음. 둘의 결합이 필수.
**C4.** 위험을 줄였는데 왜 평균수익까지 오르나(14.46→16.50)? → 레버리지 평균이 0.90<1인데도: 최악의 달들(음의 위험-수익 국면)을 미리 피하고, 좋은 국면(저변동)에 노출을 키움. Fig 3의 음(−)의 위험-수익 관계가 메커니즘.
**C5.** 왜 "더 큰 수수께끼"로 끝나나? → 위험은 낮은데 수익이 높아짐 → 위험보상 이론으로 원본보다 더 설명 불가. 저자들의 지적 정직성.
**C6.** 왜 회전율이 거의 안 느나? → σ̂(6개월 롤링)의 AR(1)=0.97로 매우 지속적 → 월별 스케일 변화가 완만.

## D. 약점 (비판적으로)

**D1.** 이론 부재 — 현상 기술·관리법 제시일 뿐, 왜 모멘텀·왜 크래시 프리미엄이 존재하는지 설명 없음.
**D2.** σ_target 12%·6개월 창은 임의적 (각주로 1·3개월 유사하다고 방어하지만 사후적).
**D3.** 거래비용은 gross 논증 — 실제 공매도 비용·증거금·차입 제약 미반영. 레버리지 2.0 실행 가능성.
**D4.** 표본 의존 — 미국 중심, 국제는 quintile·1980+로 축소. 약한 모멘텀 시장(한국·일본)에선 raw가 미미해 관리해도 절대 수익이 작음.
**D5.** 스케일 함수형(σ_target/σ̂)이 선형 역수로 고정 — 데이터가 다른 함수형을 지지할 수도(→ ML 후크).
**D6.** 하위표본 강건성은 있으나 crash 표본이 2개뿐 — 통계적으로 희귀사건.

## E. 강점 (방어적으로)

**E1.** look-ahead 제로 — 전부 t−1까지 정보. 실시간 구현 가능(Grundy-Martin의 결정적 결함 해결).
**E2.** 극단적 단순성 — 롤링 분산 하나로 샤프 2배. 복잡한 모델(GARCH·옵션·베타) 불필요.
**E3.** 국제 보편성 — 4개국+한국(우리 재현)에서 방향 일치. 일본(0.08→0.24)이 특히 강력.
**E4.** 자기자금조달 구조 활용 — 제약 없는 스케일링의 근거를 명확히 함.
**E5.** 반론 선제 방어 — 거래비용·하위표본·폭락 없는 기간까지 본문에서 처리.

## F. 활용점 (3단계 ML 논문의 후크)

**F1.** σ̂가 단순 126일 롤링 → ML 변동성 예측(HAR-RV, GBM, LSTM)으로 대체 시 성과 개선분이 기여.
**F2.** 변동성 대신 **크래시 확률**을 직접 분류(로짓/GBM) → 스케일 신호로.
**F3.** 스케일 함수 자체를 학습 — w_t = f(σ̂, 시장상태, 밸류에이션...)를 유연하게.
**F4.** 한국 세팅의 차별점: "raw가 무의미(t≈0.95)한 시장에서 ML 위험관리가 팩터를 유의하게 만들 수 있는가".
**F5.** 위험분해(Eq 7)를 ML 피처로 — 시장성분/고유성분 분리 예측.
